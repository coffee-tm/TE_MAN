# TE MAN  v2.3

首个为 ComfyUI 打造的漫剧 / 增强使用 / 生图生视频插件,为ComfyUI带来无限画布的功能特性。
包括一键分镜,一键三视图,一键故事板,自由裁切,多任务并发,资产素材库等功能。

用了新特性，只支持 Python 3.13，可去 B 站下载我的 Py3.13 整合包https://pan.quark.cn/s/e1649b85eb60


## v2.3
重磅更新:加入TE MAN 资产素材库功能,再次重塑ComfyUI使用方式
  支持一键添加ComfyUI内图片到资产库
  支持一键将资产库内图片添加进ComfyUI
  支持建立项目/资产分类
  多工作流通用,打通不同工作流项目

1. GPT image 2生图加入特调导演故事板快捷功能
2. GPT Banana Grok 生图 均加入 四视图功能
3. 增加 sora2 节点，支持 sora2 生成视频。
4. 增加 TE VEO 节点，支持 VEO 3 fast 生成视频。支持首帧,首尾帧,图参.
5. 增加 TE JM 节点，支持 JM 生成视频。
6. 修复 粘贴多出1个节点的问题

## v2.0

1. 革新 ComfyUI 提示词使用方式，完全无限画布体验，支持 @ 图片对象，极速书写。
2. 支持自动缩放图片大小，自动符合 API 尺寸像素。
3. 支持设置裁切比例 1:1、4:3、16:9 等，调整逻辑，双击裁切。
4. grok 视频支持多图参考。
5. 优化代码，提高速度。

## v1.1

1. 和 ComfyUI TE 启动器 + 特定版本配合，支持多并发 API 生图。（开发中）
2. 同一个节点同时支持输入图片、输出图片、显示图片，提高效率。（本地也可用）
3. 显示图片的同时支持显示图片提示词。（使用本插件 API 生成时）
4. 支持双击提示词复制到剪贴板。（本地也可用）
5. 支持一键三视图、四宫格图、九宫格图、导演分镜设置生成。
6. 支持一键把宫格图切割成多张图。（本地也可用）
7. 支持一键截取视频帧，截出图片。（本地也可用）
